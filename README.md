# 💫 About Me
Hi, I'm Oluwanifemi Stephen Fagbelu, a Senior Frontend Developer with expertise in building modern web and mobile applications. I hold a BSc. (Hons.) in Computer Science (Second Class Upper) from Caleb University.

Currently working as a Senior Frontend Developer at Giga Forensics, where I've launched a digital ecosystem of 5 corporate websites (Briech UAS, Bright Echefu Foundation, Giga Forensics, EIB Group, and Poctova) and architected scalable backend endpoints for a centralized task management system.

Previously, I engineered full-stack architectures at Briech UAS, built high-concurrency ticketing infrastructure at Festival Of Change, and led frontend development at TEACcH where I boosted user engagement by 100% and cut customer support inquiries by 80%.

I'm passionate about building scalable, high-performance applications and continuously expanding my full-stack expertise. My toolkit spans React, Next.js, React Native, and Electron for client-side experiences, with Express and Nest.js powering robust backends.

Ask me about React, Next.js, React Native, or full-stack architecture — I excel at translating high-fidelity designs into interactive, performant interfaces that drive business results!
